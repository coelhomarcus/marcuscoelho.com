Bem-vindo ao meu ambiente de desenvolvimento! O conteúdo estará em constante mudança, já que gosto de experimentar sempre coisas novas e sair da zona de conforto.

O tema que costumo utilizar no VS Code é o [BakaNeo](https://marketplace.visualstudio.com/items?itemName=coelhomarcus.bakaneo), um tema que eu mesmo fiz, mas ultimamente deixei um pouco de lado essa IDE.

### Ferramentas - Omarchy

`Zed` - code editor leve e minimalista, feita em Rust.

`Ghostty` - terminal moderno, focado em desempenho e com aceleração por GPU.

`Insomnia` - api client - testar, depurar e organizar requisições HTTP e GraphQL.

`DBeaver` - aplicativo completo para gerenciamento e visualização de bancos de dados.

`Zen Browser` - navegador focado em privacidade, produtividade e uma interface mais limpa.

`BleachBit` - ferramenta para limpar arquivos temporários, cache, cookies e outros dados.

### Ferramentas - macOS

`Raycast` - Spotlight com Maçã Dourada
