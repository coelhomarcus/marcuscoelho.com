Bem-vindo ao meu ambiente de desenvolvimento! O conteúdo estará em constante mudança, já que gosto de experimentar sempre coisas novas e sair da zona de conforto.

O tema que costumo utilizar no VS Code é o [BakaNeo](https://marketplace.visualstudio.com/items?itemName=coelhomarcus.bakaneo), um tema que eu mesmo fiz, mas também utilizo o Vesper++.

### Ferramentas - Windows

`VS Code` - Code editor, não preciso falar mt desse :P

`Yaak` - Api Client - testar, depurar e organizar requisições.

`DBeaver` - Aplicativo completo para gerenciamento e visualização de bancos de dados.

### Ferramentas - macOS

`Raycast` - Spotlight com Maçã Dourada

`Ghostty` - Terminal moderno, focado em desempenho e com aceleração por GPU.

`Mole` - Ferramenta que fazu um limpeza geral no seu Mac

```json
{
    "workbench.iconTheme": "symbols",
    "editor.minimap.enabled": false,
    "explorer.confirmDelete": false,
    "workbench.colorTheme": "Vesper ++",
    "js/ts.updateImportsOnFileMove.enabled": "always"
}
```
